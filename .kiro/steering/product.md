# Product Overview

A modern, lightweight productivity dashboard built with pure vanilla JavaScript, HTML, and CSS. Features a glassmorphism design with light/dark themes, complete offline functionality, and zero external dependencies.

## Core Features

- Dynamic greeting with real-time clock (24-hour format)
- Focus timer (Pomodoro) with customizable duration
- Task management with CRUD operations
- Quick links for frequently visited websites
- Light/dark theme toggle with glassmorphism design
- Complete offline functionality with Local Storage persistence

## Key Principles

- Zero external dependencies (no npm packages, frameworks, or CDN links)
- Works completely offline
- All data stored locally in browser (no server communication)
- Accessibility-first design with ARIA labels and semantic HTML
- Privacy-focused (no tracking, analytics, or network requests)
